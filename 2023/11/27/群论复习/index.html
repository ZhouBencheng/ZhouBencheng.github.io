二元运算

  • 定义： 设S为集合， 有函数f : S X S -> S的映射称为S上的二元运算, 由此可见二元运算是一种函数\映射

    • 定义解释： 得到二元运算有以下两个特点
      • 封闭性：要求二元运算必须在其作用的S集合上具备运算的封闭性，任何两个元素的运算结果必须仍在S中
      • 可操作性： 任何两个属于S的元素都必须能够参与运算， 且运算结果唯一（除法运算不能分母代零）

  • 性质：（运算六律）

    • 一种运算讨论

      • 交换律

      • 结合律

      • 幂等律： \(a * a = a\) a为幂等元

      • 消去律：

        ​ 1.从正面理解，\(a*b = a*c\) => $ b = c$

        ​ 2. \(a*b = c\)，对于运算结果c，若确定等式左边的一个操作数，则另外一个确定唯一

    • 两种运算讨论
      • 分配律
      • 结合律
      • 吸收律：
        1. 注意吸收律只在两个具有交换律的运算中讨论，不满足交换律就一定不满足吸收律
        2. 常见吸收律有交和并的运算

  • 特异元素 or 代数常数：（三种特殊元素）

    • 单位元（幺元）： 唯一性

    • 零元 ： 唯一性

      1. 在基数大于等于2的代数系统中，零元和单位元一定互异（反证法）
      2. 在不满足交换律的二元运算中，零元和单位元以及逆元需要左右讨论

    • 逆元：左逆等于右逆 => 有逆元 => 可逆

      1. 对于整个代数系统而言，单位元和零元若存在，则唯一；而逆元取决于每一个元素个体
      2. 若运算满足结合律，则元素的左逆元一定等于右逆元，逆元唯一

代数系统

  • 定义：由非空集合S和S上的二元运算f1, f2,f3, ......, fk组成的系统成为代数系统，或称为代数，记<S, f1, f2,f3, ......, fk>

有时为了强调代数常数的存在，会在代数系统记法的最后加上代数常数。在这种情况下，判断子代数，需要特地注意这类代数系统成分的存在

  • 分类：
    • 同类代数系统：1. 相同的运算个数 2. 运算操作数相同 3. 代数常数个数相同
    • 同种代数系统：在同类的基础上，按人为要求判断运算的性质是否相同

由此可见，观察一个代数系统，或者两个代数系统之间的关系，我们关注代数系统的三个成分：集合、运算、代数常数

  • 子代数：

    （判定三条件）

    1. 子集非空，且属于父代数的集合

    2. 子代数继承父代数所有运算，且子集对于所有运算封闭

    3. 被强调的代数常数需要继承

    由子代数的判定条件引入两种特殊的子代数（平凡子代数）

    • 最大子代数：父代数本身
    • 最小子代数：仅包含父代数的代数常数

  • 积代数

    定义：已知 V1 = <A, 。> 和V2 = <B, * >，得积代数V1 × V2 = <A × B, . >

    积代数系统由两个代数系统构成，这两个代数系统成为因子代数。由定义可得，积代数的集合是因子代数元素笛卡尔积的集合

​ 性质：积代数有一个很好的性质——继承性

​ 1. 运算的性质：因子代数的交换律、结合律、分配律、幂等律等运算性质都可以被积代数很好的继承

​ 2. 代数常数：单位元、零元、逆元都可以被积代数继承

综上可见积代数很好的继承了因子代数很多特殊的性质，但是注意！ 消去律是一个例外，因为积代数系统的运算是对序偶的操作，即两个元素组成的向量一并参与运算，

同态与同构

  • 同态映射定义：已知v1 = < A , 。>和 v2 = <B , * >两个代数系统，有一个映射 $ f : V1 V2 \(，且对于任意\)x,y ∈ A$
f(x 。y) = f(x) * f(y)

​ 则称 f是V1到V2的同态映射

根据定义，有一个同态映射的口诀：运算的映射等于映射的运算

  • 同态映射分类：
    1. 满同态：满射
    2. 单同态：单射
    3. 同构：双射

若是相同代数系统V到V之间的映射，则可产生自同态，自同构，单自同态，单自同构

  • 同态映射性质：

    1. 算律继承：若。运算具有交换律、结合律、幂等律等性质，那么在同态像\(f(V1)\)中的*运算也会具有此类性质，但是消去律是例外

      注意以上继承性质的讨论中，继承者必须是同态像，换言之\(Im f\)，若整个被映射代数系统能够继承性质，则该映射应该是满射，因为只有满射才能满足被映射代数系统中的每一个元素都能够参与运算

    2. 特殊代数继承：

      • $f() = $ \(\theta1\)是V1中的零元，\(\theta2\)是V2中的零元
      • \(f(e_1) = e_2\) e1是V1中的单位元，e2是V2中的单位元
      • $f(x ^ {-1}) = f(x)^{-1} $

群的定义和性质

  • 定义： 形成群的条件

tip:群一般不包含零元，因为零元不存在逆元；若一个群包含零元，则这个群只有零元这一个元素。

特殊群举例klein四元群：

  1. G中运算可交换
  2. 每个元素的逆元是本身
  3. a, b, c三个元素中任意两个元素的运算是另外一个元素

  • 群的分类

    1. 根据阶数：有限集 => 有限群； 无限集 => 无限群； 集合的基数称为群的阶数
    2. 特殊情况：只有单位元的群称为平凡群
    3. 特殊算律：群中运算满足交换律，则称为交换群或者Abel群

  • 群的性质

    1. 模n加群Zn:取Zn群中的一个元素\(a^r\) ,则该元素的阶为 \[ n \over gcd(n, r) \] 推导公式（最小公倍数定理）：$ lcm(n, r) = $

    2. \((ab)^{-1} = b^{-1}a^{-1}\)

    3. 二阶元 \(\Leftrightarrow\) 交换性

      1. 若群中的元素阶数均小于等于2，则该群是Abel群
      2. 阶数小于6的群都是Abel群

    4. 交换群(形成幂乘律：乘积的幂等于幂的乘积) $ (ab)^n = anbn$

    5. 群一定满足消去律

      证明：欲证\(ac = bc\) => \(a = b\) ,有$ acc^{-1} = bcc^{-1} $ 显然\(a = b\)

    6. $ | a | = | a^{-1}|$

      注意：对于无限阶群，其非单位元的阶可能存在，也可能不存在

    7. $ a^k = e\(当且仅当\) k|r $ ，其中\(|a| = r\)

    8. \(| b^{-1}ab| = |a|\)

    9. \(|ab| = |ba|\)

    10. 若群G为有限群，则G中阶数大于2的元素一定有偶数个

    $a G，|a| a = a^{-1} $

    1. 设$ a, b ∈ G $ ,且$ ab = ba $ , \(| a| = n, | b| = m\),且n , m 互质，则有\(|ab | = mn\)

子群和子群的陪集

子群就是群的子代数

  • 三大判定定理，证明$ H G$ **

    • （定义法）三个条件

      1. H是G的非空子集（一般子群判定中容易遗漏非空性质的判断，因此证明第一步一定要先证明单位元的包含性）

      2. $ a,b H, 有ab H$

      3. $ a H,有 a^{-1} H$

        不难发现，上述子群的定义判定法，与代数系统中子代数的判定法流程高度重合，映证开头第一句话“子群也是群的子代数”

    • （公式法）两个条件

      1. H是G的非空子集
      2. \(\forall a,b \in H,有ab^{-1} \in H\)

    • （有穷判定）两个条件

      1. H是G的非空子集

      2. $ a,b H, 有ab H$

        在有穷集合中，运算的封闭性自然而然地推导出每一个元素逆元的必然存在性

  • 特殊子群

    1. 生成子群 \(<a>\)

    2. 中心:群G中所有可交换元素构成的集合 $H = {a|a G x G(ax = xa)} $

      对于Abel群，其子群“中心”就是自己本身；而对于其他非交换的普通子群，{e}是其中心

    3. $对于群G，若有H，K G,则H K G $; 而两个子群的并集一般不再是子群

  • 陪集（定义见课本）

    • 陪集的性质

      1. $ He = H$
      2. \(\forall a \in G, a\in Ha\)
      3. 等价定理 $ aH = bH <=> a bH <=> ab^{-1} H$
      4. 划分性$ a,b H, Ha = Hb 或 Ha Hb = $

    • 等价关系

      • 定义：定义一个在群G上的二元关系R，\(H \le G\) \[ <a, b> \in R \Leftrightarrow ab^{-1} \in H \] 则该二元关系R是G上的等价关系

        定义解释：其实这个G上的二元关系换句话说，就是G中能跟H形成相同陪集的元素具有R关系

        结合上述等价定理，和此处等价关系的定义可知，对于一个子群H，不同的代表元素若能形成相同的陪集，则这些在G中的元素构成一个个等价类，也是G的划分

  • Lanrange定理

    • 定理：设群G为有限群，$ H G\(，则\)$ |G|=|H|[H : G] $$ 其中[H : G]是子群H在G中可以形成的陪集的个数，称为H在G中的指数

      根据群的消去律特性，对于群G，$ a G, aG = G$ 。同理可得，对于作为一个群的子群H，其陪集aH的基数等于H的阶数，即|aH| = |H|;又因为子群H的指数是它能够生成的陪集个数，陪集作为G的一个个划分，上述乘积自然也就是G的阶数

    • 推论1：设|G| = n，\(则\forall a \in G, |a| 是|G|的因子，且a^n = e\)

      由Lagrange定理可得，子群的阶数一定是群阶数的因子，而一个元素的阶数等于其生成子群的阶数，即 \[ |a| = |<a>| \] 所以，子群的阶 = 生成子群的阶 是 群的因子，由此得到元素的阶和群阶的关系

    • 推论2：若群G是素数阶群，则G = ,即G是循环群

      根据上一条推论知道，元素的阶一定是群阶的因子，而本题条件中，群阶是素数，那么元素的阶要么是1，要么等于群阶，而阶数为1的元素是单位元，因此该群G中一定存在一个元素是G的生成元

循环群

  • 定义：设群G中$a G,使G = $,则称G为循环群，a是循环群G的生成元

根据循环群的阶数将循环群分为两种，n阶循环群和无限循环群

  • 生成元判定：

    1. n阶循环群：对于有限阶循环群，生成元有 \(\Phi(n)\)个，对于任何一个小于n且与n互素的自然数r，\(a^r\)都是群G的生成元

      至于为什么一定是指数与n互素的元素为生成元，可以这样理解：已知有公式,对于\(a \in G\) \[ |a^r| = \frac{n}{(n, r)}, 即(a^r)^{n \over (n, r)} = e \] 这个公式直接由来显然是最大公倍数定律。回到本结论，根据公式，显然只要当(n, r) ** 1，\(a ^ r\)的阶数就是循环群阶数，自然就可以用它生成整个循环群

    2. 无限阶循环群：对于\(G = <a>\)只有两个生成元，即a和\(a ^ {-1}\)

  • 三条性质

    1. 循环群的子群还是循环群

    2. 若循环群G是无限循环群，则其子群除了{e}外，其他子群都是无限循环群

    3. 若循环群G是n阶循环群，则对于n的每一个因子d,都有对应的d阶循环子群

      对于第三条的思考，根据lagrange定理，群的每一个元素的阶都为群阶的因子，还是根据公式 \[ | a^r| = \frac{n}{(n, r)} \] 得要想获得一个d阶子群，只要取\(a^ \frac{n}{d}\)作为生成元即可
